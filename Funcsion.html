1. Основні клавіатурні команди

Дія Клавіша (Desktop/Laptop) Опис
Зупинити мовлення контрол Миттєво зупиняє мовлення
Тимчасово призупинити шіфт Призупиняє мовлення; натисніть повторно для продовження
Меню NVDA NVDA+n Відкриває меню налаштувань та інструментів
Режими озвучення NVDA+s Перемикає між мовленням, сигналами та тишею
Допомога при вводі NVDA+1 Повідомляє назву та опис будь-якої натиснутої клавіші
Завершити NVDA NVDA+q Повне завершення роботи програми
Режим сну NVDA+шіфт+s (ПК) / z (Ноут) Вимикає команди NVDA для конкретної програми

2. Навігація за допомогою NVDA (Системний фокус)

Ім'я Клавіші (Desktop / Laptop) Опис
Поточний фокус NVDA+таб Повідомляє поточну позицію; натисніть двічі для читання по символах
Заголовок вікна NVDA+t Промовляє заголовок вікна; тричі для копіювання в буфер
Вміст вікна NVDA+b Читає весь вміст активного вікна (корисно для діалогів)
Рядок стану NVDA+в кінець / шіфт+в кінець Повідомляє статусний рядок, якщо NVDA може його знайти

3. Команди для роботи в таблицях

Використовуйте контрол+алт разом зі стрілками:

  • Вліво / Вправо: перехід до попереднього/наступного стовпчика
  • Вгору / Вниз: перехід до попереднього/наступного рядка

4. Режим огляду

Переключити режими NVDA+Пробіл Між режимами огляду та редагування
Оновити документ NVDA+f5 Перезавантажує вміст сторінки або документа
Знайти NVDA+контрол+f Відкриває вікно пошуку тексту
Знайти далі/попереднє NVDA+f3 / шіфт+f3 Пошук наступного або попереднього збігу

5. Навігація за допомогою літер (Швидкі клавіші)

Натискайте літеру для переходу вперед, або Shift + літера для переходу назад:

  • h — заголовок
  • l — список
  • i — елемент списку
  • t — таблиця
  • k — посилання
  • f — поле форми
  • b — кнопка
  • x — прапорець
  • e — поле редагування
  • g — зображення
  • 1-6 — рівні заголовків

6. Налаштування (NVDA+control+k / m)

Клавіатура (NVDA+control+k)

  • NVDA+2 — Промовляти символи при введенні
  • NVDA+3 — Промовляти слова при введенні
  • NVDA+4 — Промовляти командні клавіші

Мишка (NVDA+control+m)

  • NVDA+m — Увімкнути відстеження мишки (читання тексту під курсором)
