Windows

  • Комбінація клавіш: Windows + U
  • Оповідання (Narrator): озвучує елементи на екрані
  • Екранна лупа (Magnifier): збільшує частину або весь екран (Windows + +)
  • Висока контрастність: Win + Left Alt + Print Screen
  • Вказівник миші: зміна розміру та кольору
  • Екранний диктор: Win + Ctrl + Enter
  • Фільтри кольору: Windows + U
  • Текстовий курсор: Windows + U
  • Мовлення: Windows + U

Chromebook

  • Синтез мовлення (ChromeVox): озвучує текст на екрані
  • Комбінація клавіш: Alt + Shift + S
  • Великі літери: збільшення тексту
  • Висока контрастність: покращена читабельність
  • Навігаційні кнопки: кнопки на нижній панелі

MacBook (macOS)

  • Меню: Apple → Системні параметри → Спеціальні можливості
  • VoiceOver: озвучує елементи на екрані
  • Збільшення (Zoom): масштабування екрана
  • Керування голосом: керування Mac голосом
  • Клавіатура: Sticky Keys та інші налаштування
  • Комбінація клавіш: Fn + Option (⌥) + Command (⌘) + F5
    або на моделях з Touch ID — тричі натиснути Touch ID
